# Contributing

Thanks for considering a contribution to Agent Rules Kit.

## Development

```bash
npm install
npm run build
npm test
npm run lint
```

## Guidelines

- Keep generated rules concise and practical.
- Avoid adding official-support claims for uncertain agent paths.
- Add tests for new agents, profiles, presets, or file behavior.
- Do not add AI API calls, backend services, or model credentials.
- Preserve no-overwrite-by-default behavior.

## Release Notes

Update `CHANGELOG.md` for user-facing changes.
